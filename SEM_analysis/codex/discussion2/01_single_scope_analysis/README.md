# 单口径结果解释

本目录汇总 GPP 与 RECO 在两种口径下的单独解释结果。文稿按照学术论文讨论部分的写法组织，分别结合 SHAP beeswarm、dependence plot 与 SEM 路径系数，对每个 biome 的主导机制进行定量说明。

文件包括 `gpp_prepeak_event_analysis_cn.md`、`gpp_recoverywin_analysis_cn.md`、`reco_prepeak_event_analysis_cn.md` 和 `reco_recoverywin_analysis_cn.md`。其中 `reco_recoverywin_analysis_cn.md` 使用用户指定的 `sem_recoverywin_gpp_precipEsum_vpd_hybrid_pruned_20260415` 作为过程解释口径的 SEM 来源，因此在个别 biome 与 compare 目录中的旧 SEM 版本会有数值差异。
