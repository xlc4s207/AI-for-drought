# RECO 前置预测口径单独解释分析

本文件对应 `reco_code1_flash_smrz_mswepE_clean/prepeak_event_shap_sem_20260421` 与 `sem_prepeak_event_mechanism_20260421` 的结果。与 GPP 前置预测口径类似，这一框架强调峰值前信息对后续 RECO 恢复时间的前置指示作用，但其生态含义更偏“呼吸相关的水热背景记忆”。因此，在解释时必须同时考虑目标方程中的直接效应，以及 `TMP/EVA/STRD` 经由 `VPD`、`SMrz` 与 `LAI` 传递到恢复时间的间接效应。

RECO 前置预测口径的 SHAP 平均 holdout R2 为 `0.409`，与 GPP 前置口径非常接近；但 SEM 平均 holdout R2 仅为 `0.055`，明显低于 GPP。把间接效应纳入之后可以看得更清楚：RECO 的前置信号并非缺少机制，而是大量由 `TMP → VPD → t_recover` 这样的中介链主导，导致单看直接路径很容易误判其主控因子。换言之，RECO 的前置预测框架更像“以大气需求为核心的统计-机制混合记忆结构”，其路径压缩性弱，但物理含义依然明确。

Forest 的 SHAP 结果表明 `SSRD` 和 `EVA` 是最稳定的统计控制项，但加入间接效应后，SEM 的解释重心明显转向“蒸散直接作用 + 温度经 VPD 的中介作用”。直接效应中 `EVA=+0.442` 和 `TMP=+0.336` 最强，而 `TMP→VPD→t_recover=-0.241` 是首要间接链，最终使 `TMP` 的总效应被压缩到 `+0.108`，明显低于 `EVA=+0.407`。同时，`STRD` 虽无直接入射目标方程，却可通过 `STRD→VPD→t_recover=+0.121` 形成纯中介效应。这意味着森林 RECO 的峰值前记忆主要记录了蒸散消耗本身，而温度信号更多通过改变大气水分亏缺来调节恢复尾部。

Grassland 和 Savanna 都显示出强烈的 VPD 中介特征，但两者的补偿格局不同。Grassland 中，目标方程给出 `VPD=-0.389`、`TMP=+0.211`、`EVA=+0.203`，看似温度和蒸散都会延长恢复；然而最强间接链 `TMP→VPD→t_recover=-0.478` 远强于温度直接项，使 `TMP` 的总效应转为 `-0.281`，说明草地呼吸恢复更受峰值前大气需求背景支配。Savanna 则保留了 `EVA=+0.295` 的强直接正效应，尽管 `TMP→VPD→t_recover=-0.151` 会显著抵消其温度正项，但总效应上仍以 `EVA=+0.238` 最强。这说明稀树草原 RECO 的前置记忆中，蒸散耗水仍是主轴，而温度主要通过 VPD 链条提供负向校正。

Cropland 的直接效应结构最容易产生误读。目标方程中 `TMP=+0.386`、`VPD=-0.312`、`EVA=+0.280` 同时很强，如果只看直接项，会得出“温度升高显著延长 RECO 恢复”的结论；但间接分解显示 `TMP→VPD→t_recover=-0.423` 几乎完全抵消了温度正项，使 `TMP` 的总效应收敛到 `-0.060`，反而是 `EVA=+0.106` 与 `STRD=+0.089` 成为更稳定的上游气候主导。这说明农田呼吸恢复对峰值前热量异常的响应高度依赖于 VPD 通道，不能将温度本身视为独立、单向的主控量。

Shrubland 的 RECO 前置口径呈现出“负向大气需求主导、正向植被状态补偿”的结构。直接效应中 `VPD=-0.202` 为最强负项，`EVA=+0.115` 和 `LAI=+0.070` 为主要正项；而最强间接链仍是 `TMP→VPD→t_recover=-0.134`，使 `TMP` 的总效应达到 `-0.199`。同时，`SMrz` 的总效应虽仅 `+0.041`，但其中超过一半来自 `SMrz→LAI→t_recover`，说明灌丛呼吸恢复更容易通过植被结构和底物供应状态体现水分记忆。与 GPP 相比，这里的 `PRE` 依旧不是稳定的线性主控量，说明灌丛系统对降水更可能表现为条件性响应，而不是简单单调关系。

Wetland 是 RECO 前置预测口径中统计信号最强、但结构最难压缩的 biome。其 SHAP holdout R2 达到 `0.544`，`SSRD` 的 dependence 跨度也高达 `-44.89`；然而 SEM holdout R2 仅为 `0.039`。补上间接效应后可以看到，湿地并非“没有机制”，而是主要由 `TMP→VPD→t_recover=-0.105` 与 `TMP→SMrz→t_recover=-0.025` 等多条弱到中等强度通道共同驱动，因此单一路径都不够强，但叠加后 `TMP` 的总效应仍达到 `-0.139`。这说明湿地 RECO 对峰值前辐射和温度高度敏感，但这种敏感性主要表现为多通道耦合，而不是简洁的一条线性主链。

综合而言，RECO 前置预测口径的主导图像在纳入间接效应后更加明确：Forest 与 Savanna 以 `EVA` 直接作用为主，Grassland、Shrubland 与 Wetland 则更受 `TMP→VPD` 中介链控制，Cropland 显示最强的“温度直接项与 VPD 间接项相互抵消”特征。这说明 RECO 的前置恢复记忆虽然比 GPP 更分散，却并不缺乏物理结构，其核心在于峰值前热量和耗水如何通过大气需求与植被-土壤状态共同调节呼吸回归基线的速度。
