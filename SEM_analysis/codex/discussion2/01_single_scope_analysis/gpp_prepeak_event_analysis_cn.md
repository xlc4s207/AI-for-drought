# GPP 前置预测口径单独解释分析

本文件对应 `gpp_code1_flash_smrz_onsetpeak_clean/prepeak_event_shap_sem_20260420` 与 `sem_prepeak_event_mechanism_20260420` 的结果，目标变量为 `t_recover_to_baseline_abs_peak`。这一口径强调峰值前信息对后续恢复时间的前置指示作用，因此解释重点不仅包括目标方程中的直接效应，还包括 `PRE/EVA/TMP` 通过 `SMrz`、`VPD` 与 `LAI` 写入恢复记忆的间接传递链。

|---|---:|---:|---|---|---|---|---|
| Forest | 0.434 | 0.276 | EVA, SSRD, STRD, VPD, ONS | SSRD `Δ=-28.54`；EVA `Δ=+23.34` | EVA `+0.397`；VPD `-0.128`；SMrz `-0.071` | TMP→VPD→t `-0.314` | EVA `+0.387`；TMP `-0.310` |
| Grassland | 0.439 | 0.300 | SSRD, TMP, PRE, VPD, EVA | SSRD `Δ=-19.91`；PRE `Δ=+14.32` | TMP `-0.307`；EVA `+0.304`；SMrz `+0.188` | EVA→SMrz→t `-0.108` | TMP `-0.420`；EVA `+0.182` |

从整体上看，GPP 前置预测口径的 SHAP 表现稳定，5 个 biome 的 holdout R2 集中在 `0.362-0.448`，平均值约 `0.423`；SEM 的平均 holdout R2 为 `0.220`，明显高于过程解释口径。更关键的是，把间接效应纳入之后可以看到两条稳定的中介骨架：一条是 `PRE/EVA/TMP → SMrz → t_recover` 的土壤水分记忆链，另一条是 `TMP/STRD/EVA → VPD → t_recover` 的大气需求记忆链，而 `LAI` 主要承担二级调节作用。这说明 GPP 的前置预测能力并不是若干单变量相关性的叠加，而是峰值前水热状态通过多个中介层级共同写入恢复时间。

Forest 的主导变量仍是 `EVA-SSRD-STRD-VPD`，但在补上间接效应后，机制结构比仅看目标方程更清楚。直接效应中 `EVA=+0.397` 为最强正项，而 `VPD=-0.128` 和 `SMrz=-0.071` 为主要负项；同时，`TMP→VPD→t_recover=-0.314` 是所有中介链中最强的一条，`STRD` 虽然没有直接入目标方程，但可通过 `STRD→VPD→t_recover=+0.227` 形成显著正向总效应。因此，Forest 的前置记忆可概括为“蒸散消耗增强恢复拖尾，而热量背景主要借由大气干旱通道缩短恢复时间”。这一点与 dependence 中 `SSRD` 的强负响应和 `EVA` 的强正响应是一致的，说明森林系统同时受能量供给与干燥度耦合控制。

Grassland 呈现出更典型的“热量-土壤水分”双通道结构。目标方程中的直接效应以 `TMP=-0.307`、`EVA=+0.304` 和 `SMrz=+0.188` 为主，而最强间接链并非来自温度，而是 `EVA→SMrz→t_recover=-0.108`，说明蒸散增强一方面直接拉长恢复尾部，另一方面又通过削弱土壤含水量产生相反的缩短效应。综合直接和间接后，`TMP` 的上游气候总效应达到 `-0.420`，成为真正最强的背景控制量；`PRE` 则主要以纯中介形式存在，总效应 `+0.043` 基本全部来自 `PRE→SMrz→t_recover`。因此，Grassland 的恢复记忆不是单一辐射效应，而是峰值前热量状态经水分储存过程重塑后的综合结果。

Savanna 的结果表现出更明显的补偿关系。SHAP 端 `SSRD`、`EVA`、`STRD` 与 `PRE` 同时靠前，表明前置记忆来源较为混合；SEM 中直接效应则由 `EVA=+0.351`、`VPD=-0.068` 和 `SMrz=+0.065` 主导，而最强间接链是 `TMP→VPD→t_recover=-0.112`。也就是说，温度升高并不完全通过直接路径起作用，而是更多借助 VPD 通道压缩恢复时间；相反，蒸散增强的正向直接作用更强，因此 `EVA` 的总效应仍维持在 `+0.308`。这一结构说明稀树草原中“蒸散耗水拉长恢复”和“热力驱动干燥加快恢复”是并存的，dependence 图中 `SSRD` 的快速下降和 `EVA` 的上升拖尾正好对应了这种双向竞争。

Cropland 在只看 SHAP 排序时容易被解释成 `SSRD-STRD-EVA-VPD` 主导，但 SEM 的效应分解揭示出更隐含的温度控制。目标方程中 `EVA=+0.261` 与 `TMP=-0.244` 分别代表最强正、负直接项，`SMrz=+0.128` 说明较湿润根区背景会延长回到基线的时间；同时，`EVA→SMrz→t_recover=-0.084` 与 `TMP→SMrz→t_recover=-0.052` 显示，蒸散和温度都可先通过耗水削弱 SMrz，再反向缩短恢复时间。于是，在总效应尺度上，`TMP=-0.294` 反而超过 `EVA=+0.176` 成为最强的上游气候因子。这说明农田系统的前置预测记忆并不只是能量放大本身，更是峰值前热量和耗水通过根区水分中介共同塑造的结果。

Shrubland 是 GPP 前置口径中最具“非一致性”的 biome。SHAP 中 `PRE` 与 `VPD` 均很靠前，dependence 也提示湿润背景和大气干燥都可能显著改变恢复时间；但 SEM 分解显示，目标方程的主导直接项是 `TMP=-0.323`、`EVA=+0.109` 和 `SMrz=-0.066`，而 `TMP` 的正向间接链 `TMP→VPD→t_recover=+0.047` 又会部分抵消其负向直接效应，使其总效应收敛到 `-0.284`。与此同时，`EVA` 的总效应增至 `+0.127`，说明蒸散既有直接拉长恢复的作用，也通过土壤水分链条放大这一趋势。值得注意的是，`PRE` 在 SEM 中仅表现为很弱的净负中介效应，而在 SHAP 中却有显著的重要性，表明灌丛系统中降水影响更可能以非线性阈值或与其他背景变量的协同方式出现，而不是稳定的线性主路径。

总体而言，GPP 前置预测口径的物理图像在纳入间接效应后更加完整：Forest 与 Savanna 主要体现为 `EVA` 的直接拉长效应与 `TMP→VPD` 的负向中介并存，Grassland 与 Cropland 则显示出更强的 `SMrz` 水分储存链，Shrubland 具有最明显的路径抵消与非线性迹象。这意味着该口径不仅适合作为恢复难度的前置预测框架，也能为不同 biome 中“能量背景、耗水过程与根区水分记忆如何耦合”提供机制解释。
