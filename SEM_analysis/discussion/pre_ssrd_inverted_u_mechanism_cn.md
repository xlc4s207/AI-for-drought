**核心现象**：PRE 与 SSRD 的 SHAP 都呈近对称的倒 U 形；两端为负（该特征取值推低预测恢复时间），中间为正（推高）。两侧非零起点位置在各 biome 内部有很强的复现性：PRE 峰值集中在 `0.003–0.005 m/day`（Shrubland 向低端偏移至 ~0.001），SSRD 峰值集中在 `190–230 W/m²`。

## 2. 为什么会出现倒 U：两条机制耦合在一起

t_recover 是被预测的目标变量，SHAP > 0 表示该特征在该取值水平上把恢复时间“推得更长”，SHAP < 0 则把恢复时间“推得更短”。因此倒 U 的每一段对应的是**样本所处水气候状态不同**，而不是该变量本身对 GPP 的单调生理效应。分箱共变量（表 A1–A2，见下）给出了各段落的主导背景。

### 2.1 PRE 倒 U 形的三段式机制

**下行起点（极低 PRE，SHAP 强负）**
- 共变量：高 VPD（Cropland 16，Grassland 13.5，Shrubland 15.9）、低 SMrz、低 LAI、高风速、高 SSRD，空气温度相对偏低。
- 物理含义：这是**干适应/半干旱背景样本**。事件发生时可供蒸腾的水本来就少，恢复期降水接近于零并不是制约恢复的瓶颈——水分限制已经从外部（降水）转移到内部（土壤背景水+植被量），真正的瓶颈不是 PRE。此时事件的观测 t_recover 普遍很短（系统缺乏继续恢复的物质基础，或 baseline 本身就低），于是模型把“极低 PRE”与“短恢复”关联起来，产生强负 SHAP。
- 结论：下行端反映的是**采样结构效应**，不是“越少雨恢复越快”。

**上行段（亚阈值 PRE，SHAP 由负转正）**
- 共变量：SMrz、LAI 随 PRE 同步上升，VPD 从高位下降到 5–10 hPa，温度与辐射也同步下调。样本从“干背景快恢复”过渡到“较湿润、植被活跃、恢复期较长”的事件。
- 物理含义：这些样本的 PRE 数量级刚好位于“能够接收到、但还不足以完全补水”的门槛区间。PRE 仍是恢复的限速因子，而此时 t_recover 的观测值相对长，模型把“中等 PRE 水平”与“长恢复”关联起来，SHAP 转为强正。
- 结论：上行段主要是**样本成分切换 + PRE 处于限水阈值附近**两者叠加的结果。

**下行段（超阈值 PRE，SHAP 再次转负）**
- 共变量：VPD 继续下降（Forest 2.8、Cropland 4.5）、SMrz 达到最高（Forest 0.38、Cropland 0.36）、LAI 最大，而 SSRD、wind 与 T2m 都随之走低。样本主要是**湿润/多云、背景水分充足**的事件。
- 物理含义：PRE 已经越过有效补水阈值（约 `0.004–0.006 m/day`，即 4–6 mm/day），额外的降水不再是“结构指示物”，而是真正供水给系统的增量水分；同时高 PRE 时段往往伴随低辐射/低 VPD，大气需求被压低，恢复被加速。
- 结论：下行段代表**水分供给饱和 + 大气需求降低的协同加速**，这才是文献中“降水促进恢复”的经典情景。
- 唯一例外：Shrubland 与 Grassland 高 PRE 端 SHAP 仅弱正/回零而非深负，原因是这两个 biome 的样本几乎不存在真正的高 PRE 子集（PRE>8 mm/day 占比仅 1.4% 与 2.7%）——统计上就走不到“下行段的尽头”。

### 2.2 SSRD 倒 U 形的三段式机制

SSRD 与 VPD、T2m、SMrz 在同一 biome 内的 Spearman 相关极强（VPD：+0.79 … +0.88；T2m：+0.58 … +0.82；SMrz：−0.11 … −0.61）。SSRD 的 SHAP 曲线因此不能单独当作“光量对恢复的效应”来读，而是**一条被共变大气需求绑在一起的样本轨迹**。

**低 SSRD 段（SHAP 负）**
- 共变量：VPD 很低（Forest 1.3、Savanna 3.0）、SMrz 高（Forest 0.44、Savanna 0.38）、T2m 低（279–285 K）、PRE 中高、LAI 高。
- 物理含义：这是**多云—湿润—冷凉**的复合背景。大气蒸腾需求被抑制，土壤水不缺，整段事件的恢复比“同分区平均”更轻松，因此在这个 SSRD 取值下 t_recover 偏短，模型给出强负 SHAP。
- 读作“光照少反而恢复快”是误读——真正在做功的是同时出现的低 VPD、高 SMrz。

**中 SSRD 段（SHAP 正峰值）**
- 共变量：VPD 开始上升到 6–12 hPa，SMrz 仍中等，温度 290–297 K。
- 物理含义：中等辐射水平恰好是**“供水能跟上但大气需求开始加码”的过渡带**。事件通常发生在生长季中期，t_recover 更长（因为植物要重新铺开 LAI、重建碳平衡），模型把“中等 SSRD”关联到“长恢复”，SHAP 达到正峰。
- PRE 在这一段往往也处于其自身的限水阈值附近，两条倒 U 的正峰在同一组事件上共振。

**高 SSRD 段（SHAP 负）**
- 共变量：VPD 极高（Cropland 19.6、Grassland 20.6、Shrubland 22.0）、SMrz 最低（Grassland 0.079）、LAI 下降、PRE 接近零、温度 297–300 K、风速上升。
- 物理含义：这是**晴空—干热**的复合背景。表面看是“强辐射样本”，实质是**大气需求—土壤水同时被推到极端**的干旱样本。此时：① 事件往往无法在分析窗口内完成完整恢复，t_recover 被截断或不完整；② 在 Shrubland/Grassland 等干适应系统里，恢复本身就很短（“能长多少就长多少”就停了）；两种机制都让 t_recover 偏短，强辐射于是被关联到短恢复，SHAP 转负。
- 与 PRE 的下行段不同：这里 t_recover 偏短**不是**“恢复得好”，而是“恢复得不充分或生态容量耗尽”。这是 SSRD 下行段必须在论文中明确区分的关键。

### 2.3 为什么两者会共同呈倒 U？——共变量带动的联合分布

1. 水气候联合分布：PRE、SSRD 的两端在大尺度上分别对应两类极端水气候——“干旱背景”（低 PRE / 高 SSRD 两端共享）与“多云—湿润”背景（高 PRE / 低 SSRD 两端共享）。
2. 事件持续时间分布：两端的事件持续时间分布系统性偏短（干背景下事件本来就短；湿润多云背景下恢复本来就快），而中间段落集中了 t_recover 偏长的事件。
3. 模型归因：LightGBM-SHAP 的条件归因会把“取值落在中段”这一事实本身当作“长恢复”的信号，反映在 SHAP 曲线上就是中间段正、两端负的倒 U。

换言之：**这两条倒 U 并不是“降水/辐射先促进后抑制 GPP 恢复”的直接生理响应，而是水气候—样本结构—事件时长的联合分布通过 SHAP 条件归因投射出来的统计指纹。**

## 3. 面向论文撰写的建议框架

### 3.1 结果段推荐写法

> 在五个主要植被类型中，恢复期平均降水（PRE）与其 SHAP 值均呈现一致的先升后降的倒 U 形关系（图 x）。SHAP 峰值对应的 PRE 集中在 `3–5 mm/day`（Shrubland 更低，约 `1 mm/day`），峰值 SHAP 约为 `+6 ~ +11` 天；当 PRE 低于 `0.5 mm/day` 时 SHAP 降至 `−19 ~ −24` 天，当 PRE 超过 `8 mm/day` 时 SHAP 再次回落至负值。恢复期平均短波辐射（SSRD）呈相同形态的倒 U，SHAP 峰值对应的 SSRD 集中在 `~190–230 W/m²`，峰值 SHAP 约为 `+3 ~ +9` 天；两端的 SHAP 分别降至 `−4 ~ −12` 天。两条倒 U 的中间正峰彼此相关（SHAP 尺度的 Spearman 相关 ρ = `+0.13 ~ +0.47`），提示两者由共同的水气候—事件样本结构驱动。

### 3.2 讨论段推荐写法

> PRE 和 SSRD 的 SHAP 倒 U 形并非“降水/辐射对 GPP 恢复先促进后抑制”的直接物理响应，而应解释为**水气候—样本结构—事件时长三重耦合**在条件归因上的投影。就 PRE 而言，左侧下行端对应的是极端干旱背景下**PRE 根本不是恢复的限速因子**的样本（SMrz、LAI 本底偏低，事件持续时间短，因此 t_recover 短）；中段上行—峰值区对应的是**PRE 处于 `3–5 mm/day` 的有效补水阈值附近**但仍不足以完全解除水分限制的事件（样本背景水分增加、LAI 变大、恢复期变长）；右侧下行端则对应**PRE 越过补水阈值、同时 VPD/辐射需求被压低**的真正促进恢复情形。就 SSRD 而言，左端的负 SHAP 来自**多云—湿润—冷凉复合背景**，并不是“光照少直接加速恢复”，而是共出现的低 VPD、高 SMrz 共同作用；右端的负 SHAP 来自**晴空—干热复合背景**下恢复被大气需求与 SMrz 亏空截断或在干适应系统中本身就短，因此表现为“高辐射对应短 t_recover”。将两条机制归并，可以得到一条统一的解释：**在水气候光谱的两端，事件样本的 t_recover 分布系统性偏短；中段则集中了长恢复事件；模型通过条件归因把中段取值识别为“长恢复信号”，因而呈现出 PRE 与 SSRD 的联合倒 U**。因此不宜将图中的“先升后降”解读为单变量的非线性生理响应，而应理解为水气候联合状态与事件时长分布的结构性映射；真正的供水加速作用体现在 PRE 右侧下行段里 `VPD↓、SMrz↑、LAI↑` 同时出现的湿润样本子集，而真正的辐射胁迫作用体现在 SSRD 右侧下行段里 `VPD↑、SMrz↓、PRE≈0` 同时出现的干热样本子集。

### 3.3 可在文中保留的关键量化证据

- PRE 峰值位置（biome 自 `0.003–0.005 m/day`，Shrubland `~0.001 m/day`）与 SSRD 峰值位置（`190–230 W/m²`）。
- 5000 样本内 **Spearman 共变结构**：ρ(SSRD, VPD) 在各 biome 均 > `+0.79`；ρ(SSRD, SMrz) 在 Forest/Grassland 达 `−0.35 ~ −0.61`；ρ(PRE, VPD) 在各 biome 均 ≤ `−0.17`；ρ(PRE, SMrz) 在各 biome 均 ≥ `+0.22`。
- 两端样本占比证据：Grassland/Shrubland 的 `SSRD>2.4×10⁷ J/m²/day` 样本占比约 `27%`，对应干热极端背景；Shrubland 的 `PRE<10⁻⁴ m/day` 样本占比 `24.6%`。这些占比支持“两端分布由极端水气候构成、而不是绘图伪影”的说法。
- 各 biome 倒 U 的**非对称性**：Shrubland/Grassland 的右侧下行支不明显（几乎没有高 PRE 样本）；Cropland/Forest/Savanna 的右侧下行支最完整。论文中可以按 biome 标注这一差异，避免“所有 biome 曲线都一样”的过度概括。

## 4. 仍需注意的边界（审稿人可能问到）

1. SHAP 的倒 U 不能作为因果结论；更严格的证明需要 **分层条件分析**（固定 VPD、SMrz、T 分位数后重新做 PRE / SSRD 的响应曲线），或者进行 **partial dependence + ALE 对比**。
2. 目标变量 `t_recover_to_baseline_abs_peak` 在高 SSRD/低 PRE 的干旱样本里可能存在 **事件未闭合（右删失）** 的情况，这会进一步压缩这些样本的 t_recover，从而放大下行端的负 SHAP。论文中建议加一句“对未能在分析窗口内回到基线的事件采用的截断/处理规则”作为 caveat。
3. PRE 与 SSRD 之间在样本尺度上存在弱负相关（ρ ≈ `−0.05 ~ −0.32`），因此“两者的倒 U 不是同一个倒 U”——它们各自捕获了水气候联合分布的一个主轴。写作时不要把两者合并为一条“水热协同倒 U”，而应该并列说明。
4. Shrubland 的 PRE 峰值明显偏左（`~0.001 m/day`）是一个独立信号：干适应系统的有效补水阈值远低于湿润系统。这一点可以作为一个次级结论单独写。
