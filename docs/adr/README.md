# Architecture decision records

Use this directory for durable research workflow and repository-management decisions.

Create records as:

```text
YYYY-MM-DD-short-title.md
```

Recommended template:

```md
# Decision: <title>

## Status

Proposed | Accepted | Superseded

## Context

What problem or constraint led to this decision?

## Decision

What did we choose?

## Consequences

What becomes easier, harder, or risky?

## Validation

How will we know this decision is working?
```
